<h2>德园</h2>

1、进门处的面条，酸笋肉丝面、红烧牛肉面都还可以

2、进门处的大包也还行，叉烧包味道不错

3、早上可以去吃豆腐脑，物美价廉

4、烤鸭也还可以，鸭头和鸭脖很便宜
